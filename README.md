# UTS-PAM
### Sebuah aplikasi pemutar video 

### Daftar package yang digunakan (expo)
 - expo-asset
 - expo-av,
 - expo-splash-screen
 - expo-status-bar
 - expo-updates

### Cara menginstall aplikasi (dalam mode pengembangan)

1. install dependensi yang diperlukan
```sh 
npm install
```
2. install expo
```sh 
npm i expo
```
3. build to apk (android)
```sh 
expo build:android
```
